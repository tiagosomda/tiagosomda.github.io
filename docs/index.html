Tech Interests

XR (eXtended Reality):

This domain is the tech I am most passionated about it. I can see myself retiring wearing an XR headset (or lenses!).

Decentralization:

There are cons and pros with decentralized systems. However, I personally see more pros than cons. My favorite initiatives are Mastodon, Solid.mit.edu / Inrupt. Blockchain is foundation for it, but as you can see with Solid and Inrupt, not everything needs to be on a blockchain.

Artificial Intelligence:

AI is changing the world. We need to make sure that we create AIs in an ethical and open way so that everyone can benefit from the power that this technology can bring. There will be private AI entities, but like Linux, the foundation should be open to all.
